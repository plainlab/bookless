# Bookless

> A no-hassle book-writing app that let you write your books in markdown.

## Features

- [x] Local markdown files edit
- [x] Focus mode
- [x] Drag & drop to organize chapters
- [ ] Spelling check
- [ ] Book & chapter preview
- [ ] PDF / ePub / DOCX export
